# Image Assets

This directory contains all image assets used in the component gallery.

## Directory Structure

```
img/
├── index/          # Images for the home page components
└── signs/          # Images for the signs gallery and content
```

## Required Images

### img/index/
- `background-building.jpg` - Background image for the story hero section
- `logo.svg` - Yoder Graphics logo
- `badge.svg` - Veteran owned badge
- `cargo-van.png` - Van image for orange feature section
- `print-card.jpeg` - Print service card image
- `creative-card.jpeg` - Creative service card image
- `promo-card.jpeg` - Promo service card image
- `fulfillment-card.png` - Fulfillment service card image

### img/signs/
- `affinitygaming.webp` - Gallery image
- `akron_childrens.webp` - Gallery image
- `bballjewelers2.webp` - Gallery image
- `bejeweled.webp` - Gallery image
- `candytyme.webp` - Gallery image
- `community.webp` - Gallery image (also used in content box)
- `securitec.jpeg` - Illuminated signs content box image

## Note

These images are currently hosted externally at `https://yg-site-preview.netlify.app/img/`.
Please download and place them in the appropriate directories listed above.
